# Sample Company Policy Document

This is a placeholder document. Please add your own PDF policy documents for the AI Policy Copilot to index and query.

## How to Use

1. Upload PDF files using the sidebar
2. Click "Index Documents" to process them
3. Ask questions in the chat interface

## Supported Document Types

- HR Policies
- Standard Operating Procedures (SOPs)
- Compliance Documents
- Employee Handbooks
- Guidelines and Procedures

## Tips for Best Results

- Use text-based PDFs (not scanned images)
- Ensure documents are in English
- Break very large documents into smaller sections
- Use clear, descriptive filenames
